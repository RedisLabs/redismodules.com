Redis Modules Hub

Redis Modules are add-ons to Redis which extend Redis to cover most of the popular use cases for any industry. They seamlessly plug into ​Redis, processed in-memory and provide Redis’ simplicity, super high performance, scalability and high availability.

Modules can be created by anyone. Similarly to the Open Source Redis Modules page, the Modules Hub features OSI licensed modules created by Redis Labs and others.

The Modules Hub is an open source project that's 100%-powered with Redis and Redis Modules.
