Redis Modules Hub

Redis Modules are add-ons to Redis which extend Redis to cover most of the popular use cases for any industry. They seamlessly plug into ​Redis, processed in-memory and provide Redis’ simplicity, super high performance, scalability and high availability.

Modules can be created by anyone. The Module Hub includes modules created by Redis Labs as well as by others.

Modules displaying the "Certified module" icon are certified by Redis Labs to be compatible both with the open source as well as the enterprise versions of Redis.

Modules quickstart

Modules quickstart

Generally, modules have their own documentation that explains how to use them. To start using most open source modules with open source Redis, follow these steps:

  1. Clone or download the source code from the module's repository
  2. Follow the instructions in the repository for building the module (usually just running the make command)
  3. Load the module to the Redis server with the loadmodule configuration directive or the MODULE LOAD
  4. Use any Redis client to connect to the server and call the module's commands

Modules that are included with redise only need to be activated for a database to be used. Refer to the redise documentation for more details.
